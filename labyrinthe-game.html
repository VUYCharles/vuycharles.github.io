À propos du projet

About the Project

Ce projet Java implémente plusieurs algorithmes classiques de génération et de résolution de labyrinthes. L'application permet de visualiser étape par étape le fonctionnement des algorithmes et de comparer leurs résultats grâce à une interface Swing.

This Java project implements several classic maze generation and solving algorithms. The application allows step-by-step visualization of the algorithms' operation and comparison of their results through a Swing interface.

Fonctionnalités

Features

Algorithmes de génération

Generation algorithms

Implémente plusieurs méthodes : exploration exhaustive, backtracking, Prim randomisé, Kruskal, division récursive.

Implements several methods: exhaustive search, backtracking, randomized Prim's, Kruskal's, recursive division.

Algorithmes de résolution

Solving algorithms

Plusieurs approches implémentées : A*, Dijkstra, recherche en profondeur, recherche en largeur.

Several approaches implemented: A*, Dijkstra, depth-first search, breadth-first search.

Visualisation étape par étape

Step-by-step visualization

Possibilité de voir l'exécution des algorithmes en temps réel avec contrôle de la vitesse.

Ability to see algorithm execution in real-time with speed control.

Interface Swing

Swing interface

Interface graphique intuitive avec personnalisation de la taille du labyrinthe et sélection des algorithmes.

Intuitive graphical interface with maze size customization and algorithm selection.
