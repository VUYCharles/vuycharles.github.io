Hyperbolic/Euclidean Comparison

À propos du projet

About the Project

Ce projet explore les différences fondamentales entre les géométries hyperbolique et euclidienne à travers une application interactive. Il permet de visualiser comment les propriétés géométriques (comme les droites parallèles, les angles et les distances) se comportent différemment dans ces deux cadres mathématiques.

This project explores the fundamental differences between hyperbolic and Euclidean geometries through an interactive application. It visualizes how geometric properties (such as parallel lines, angles, and distances) behave differently in these two mathematical frameworks.
